# Utility Scripts

## Available Scripts

### setup_kaggle.py
Automates Kaggle API setup and credential configuration.

```bash
python scripts/setup_kaggle.py
```

### download_models.py
Downloads required model weights for local development.

```bash
python scripts/download_models.py
```

### seed_db.py
Seeds the database with sample data for testing.

```bash
python scripts/seed_db.py
```

## Usage

All scripts should be run from the project root directory:

```bash
# From project root
python scripts/<script_name>.py
```
